# 探针部署方式：

1. 前往 [新建应用](http://www.webfunny.cn/webfunny/createProject)  

2. 根目录下执行命令$: webpack
   得到一个压缩js文件(探针) lib/monitor.fetch.min.js
  
    
如果有问题，欢迎提问。
